# 🌊 FlowSpace: The Minimalist Workspace Toolkit

> **Status:** 🚧 In Progress

**FlowSpace** is a full-stack, flexible dashboard application designed with minimalism in mind to help freelancers and small businesses efficiently manage their workspace and day-to-day operations.

## ✨ Features
- **Dynamic Dashboard:** Customizable, drag-and-drop workspace dashboard for tracking daily operations.
- **Cloud Sync:** Secure authentication and database management synced to the cloud via Supabase.
- **Offline Capabilities:** Built as a Progressive Web App (PWA) featuring robust local offline storage.
- **Containerization:** Clean, fully containerized full-stack architecture for easy deployment.

## 🛠️ Tech Stack 
**Frontend:**
- Next.js & React
- Tailwind CSS

**Backend / Database:**
- Python (FastAPI)
- Supabase (PostgreSQL & Auth)

**Infrastructure:**
- Docker & Docker Compose
